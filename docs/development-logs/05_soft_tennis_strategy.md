# Soft Tennis Strategy

## ソフトテニス市場の重要性

ソフトテニスは日本の中学校・高校の部活動で広くプレーされている競技です。特に中高生の競技人口、学校単位の活動、顧問や外部コーチによる指導、保護者のサポートという文脈では、硬式テニスとは異なる利用シーンがあります。

CourtLens は、一般的な硬式テニス分析アプリの単なる日本語版ではなく、ソフトテニス部活動の練習、試合、チーム指導に合わせたプロダクトとして差別化します。

## 硬式テニスとの差異

### 前衛・後衛ポジション

ソフトテニスのダブルスでは、前衛と後衛の役割が明確です。

- 前衛: ネット付近でボレー、スマッシュ、ポーチ、プレッシャーを担当する。
- 後衛: ベースライン付近でラリーを作り、相手後衛との打ち合いから展開を作る。

硬式テニスのダブルスにもネットプレーはありますが、ソフトテニスでは部活動指導上、前衛/後衛の役割分析がより重要になります。

### 雁行陣

雁行陣は、後衛がベースライン、前衛がネット付近に立つ基本形です。

分析観点:

- 後衛がクロスラリーを安定して作れているか。
- 前衛が相手後衛へプレッシャーをかけられているか。
- 前衛がポーチに出るタイミングを逃していないか。
- 後衛の返球が前衛の決定機を作っているか。

### 並行陣

並行陣は、ペアがともにネット寄りまたは同じラインで攻撃的に構える形です。

分析観点:

- ペア間の距離。
- センターカバー。
- ロブ対応。
- ボレーの決定率。
- 相手を押し込んだあとの詰め方。

## 現在の実装状況

- `SportType` に `softTennis` を定義済み。
- `TennisSession` は `HardTennisSession | SoftTennisSession` の discriminated union。
- `SoftTennisSession` に `position?: SoftTennisPosition` を定義済み。
- 現在の `SoftTennisPosition` は `forehand` / `backhand` のサイド情報として定義済み。
- 新規セッション作成とプロフィール設定でソフトテニスを選択可能。
- ソフトテニスのサンプルセッションを用意済み。
- コート寸法は硬式テニスと同一として扱う。
- コート表示ではソフトテニス用の表示色とラベルを分けている。
- レポート画面では、ソフトテニスのダブルス時に前衛コーチングの将来実装枠を表示している。

## Phase 4 以降での前衛コーチング実装計画

現時点では前衛/後衛の専用コーチングは未実装です。次の段階では、以下の順に実装します。

1. データモデル拡張
   - `SoftTennisRole`: `front` / `back` を追加。
   - `formation`: `parallel` / `staggered` / `unknown` を追加。
   - `netAction`: `poach` / `volley` / `smash` / `block` などを追加。

2. 入力 UI 拡張
   - ポイント入力に前衛/後衛、陣形、ポーチ有無、前衛タッチ有無を追加。
   - 入力負荷が増えすぎないよう、ソフトテニスのダブルス時だけ表示する。

3. 分析ルール追加
   - 前衛タッチ率。
   - 前衛決定率。
   - ポーチ成功率。
   - 後衛ラリー継続率。
   - 前衛が絡まない失点パターン。

4. レポート表示
   - 前衛/後衛別の課題。
   - ペア戦術の改善コメント。
   - 部活練習向けのメニュー提案。

## ダブルス特有分析の将来対応

### 前衛決定率

前衛が触ったポイントのうち、得点につながった割合を計算します。

活用例:

- 前衛の決定力評価。
- スマッシュやボレーの練習優先度判断。
- 後衛が作ったチャンスを前衛が決められているかの確認。

### ポーチ成功率

ポーチに出たポイントのうち、得点または相手のミスにつながった割合を計算します。

活用例:

- 出るタイミングの良し悪し。
- クロスラリー中の予測力。
- 前衛のプレッシャーの有効性。

### ペア連携分析

将来的に動画解析を組み合わせ、以下を分析します。

- 前衛と後衛の距離。
- センターの空き。
- ロブに対する入れ替わり。
- 前衛のフェイク動作。
- 後衛の配球と前衛の決定機の関係。

## 差別化の方向性

CourtLens は、ソフトテニスを後付け対応にしない方針です。MVP では型、セッション、プロフィール、コート表示、サンプルデータまでを整え、v1 以降で前衛/後衛、陣形、ポーチ、部活動チーム管理を深掘りします。

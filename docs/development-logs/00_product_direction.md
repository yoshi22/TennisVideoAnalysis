# CourtLens Product Direction

## アプリ概要

CourtLens は、硬式テニスとソフトテニス向けの動画分析アプリ MVP です。

v0 では、試合や練習セッションを作成し、動画を添付または撮影しながら、ポイントごとの結果、ショット種別、ラリー数、打球位置を手動で記録します。記録されたデータからサーブ成功率、得点率、平均ラリー数、ショット内訳、コート上の傾向、弱点、練習メニュー、コーチングコメントを確認できます。

## ターゲットユーザー

- 中高生の部活動プレイヤー: 練習試合や日々の練習で、自分の課題を継続的に把握したいユーザー。
- コーチ、顧問、外部指導者: 選手へのフィードバックを、動画とスタッツに基づいて整理したいユーザー。
- 保護者: 子どもの練習状況や試合内容を、数値とコメントで把握したいユーザー。
- 社会人プレイヤー: 草トーナメントやサークル練習の振り返りを効率化したいユーザー。

## 提供価値

- セッション管理: 試合、サーブ練習、ストローク練習、ボレー練習、自由練習をセッション単位で保存する。
- ポイント記録: 得点/失点、サーブ結果、ショット種別、結果理由、ラリー数、ショット位置を素早く記録する。
- スタッツ集計: 1st サーブ成功率、2nd サーブ成功率、ダブルフォルト数、エース数、得点率、平均ラリー数を自動集計する。
- コーチングコメント: 手動入力データから弱点パターンを検出し、改善コメントと練習メニューを提示する。
- コート可視化: 正規化座標で記録したショット位置を、硬式/ソフトテニスのコート表示に重ねる。

## 差別化方針

- ソフトテニス対応: MVP の型と UI で `softTennis` を第一級の競技種別として扱う。
- 日本市場向け: 部活動、練習試合、保護者共有、顧問コーチの利用を想定した日本語 UI を優先する。
- 部活向け: 高価な専用機材に依存せず、スマートフォンで撮影、記録、振り返りを完結できる構成にする。
- ダブルス対応構造: `matchFormat` に `singles` / `doubles` を持たせ、将来の前衛/後衛、ポーチ、ペア連携分析へ拡張できるデータモデルにする。
- ルールベースから AI へ拡張: v0 は ManualAnalyzer による手動入力とルールベース分析、v1 以降で Computer Vision と LLM を段階的に導入する。

## v0 スコープ

- Expo Router によるホーム、履歴、設定、セッション作成、ログ、コート、レポート画面。
- 硬式テニス/ソフトテニス、シングルス/ダブルスのセッション作成。
- 動画撮影、ライブラリ選択、動画 URI のセッション保存。
- ポイントログ入力フォーム。
- ショット位置のコートタップ入力。
- セッション履歴と削除。
- プレイヤープロフィール保存。
- Zustand persist と AsyncStorage によるローカル永続化。
- ManualAnalyzer によるスタッツ計算、弱点検出、コーチング Tips、練習メニュー生成。
- Jest による ManualAnalyzer の基本検証。

## v1 以降のスコープ

- Computer Vision によるボールトラッキング。
- ショット種別の自動分類。
- サーブ速度推定。
- 姿勢推定によるフォーム解析。
- 自動ハイライト生成。
- LLM によるパーソナライズドコーチングコメント。
- ソフトテニスの前衛/後衛ロール、雁行陣、並行陣を考慮した分析。
- ダブルスの前衛決定率、ポーチ成功率、ペア連携分析。
- コーチ向けの複数選手管理。
- Supabase や Firebase へのデータ同期、共有、バックアップ。
